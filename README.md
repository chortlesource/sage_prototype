# sage

A simple tower defence game written in C++

## Description

sage aims to be a tower defence game written in C++ using the SDL library.

## dependencies

* libsdl2       >= 2.0.9
* libsdl2_image >= 2.0.0
* libsdl2_ttf   >= 2.0.0
* cmake         >= 3.1
